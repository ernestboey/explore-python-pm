## Usage

1. Install dependencies
```
$ pipenv install requests
```

2. Run code

```
$ pipenv run python main.py
```

Virtualenv location was **~/.local/share/virtualenv/pipenv-asdf1234**

3. Remove virtualenv
```
$ pipenv --rm
```
